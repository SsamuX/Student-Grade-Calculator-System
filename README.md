# Student Grade Calculator System

## Project Description
The Student Grade Calculator System is a simple Python project that allows a user to enter a student's name and marks for different subjects. The program then calculates the total marks, average score, grade, and displays a performance message.

This project is useful for learning basic Python programming concepts such as:
- Variables
- Input and output
- Arithmetic operators
- Decision structures
- Functions
- Modules

## Features
- Accepts student name
- Accepts marks for subjects
- Calculates total marks
- Calculates average marks
- Assigns a grade
- Displays a performance message
- Shows the final results clearly

## Subjects Used
The system uses the following subjects:
- Mathematics
- English
- Science
- Kiswahili
- Social Studies
